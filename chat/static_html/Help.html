Command list whitch can be used in the chat
/help /? [ Command ] Displays help document.
/fullhelp Displays list of all available commands with informations about them.
/bug /suggest Send a bug report.
/room { Room name } Changes the room.
/clear Clears the current chat window.
/ignore {-add | -del } { Username }
-add => Add a user to your ignore list. That user cannot send you private messages and you can not see messages they send.
-del => Remove a user from your ignore list. That user can now send you private messages and you can see messages they send.
/private /prv { Username } Allows you to send private messages.
/removefriend /removebuddy { Username } Removes a friend from your friend list.
/friend /buddy {-add | -del | -show } [ Username ]
-add => Adds a friend from your friend list.
-del => Removes a friend from your friend list.
-show => Displays all users on your friend list.
/url { URL address } Allows you to insert a URL into the chat.
/unignore { Username } Remove a user from your ignore list. That user can now send you private messages and you can see messages they send.
/kick { Username } [ Reason ] Kick a user from the system. You may specify an optional reason. This command can only be used by administrators or operators.
/myprofile /me Informations about user.
/whoami Displays user nick.
/ping { Host } sending icmp_echo_request packages to network hosts.
/logout /exit Chat exit.
/configrooms /crooms Rooms configuration.
/skin {-setskin | -setcss | -showskins | -showcss } [ skin_name | css_name ]
-setskin => Sets skin for the chat
-setcss => Sets css for the chat
-showskins => Shows a list of available skins
-showcss => Shows a list of available css
/whois /info /profile { Username } Displays information about the specified user.
/emoticons /emot List of available emoticons.
/config Chat variables configuration
/avatar Selection of avatar for user.
/configreg Registration fields configuration.
